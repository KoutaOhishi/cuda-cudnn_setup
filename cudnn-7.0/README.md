このURL （https://developer.nvidia.com/rdp/cudnn-archive） からcudnnをインストール  

Download cuDNN v7.0.5 (Dec 5, 2017), for CUDA 9.0の  
"cuDNN v7.0.5 Runtime Library for Ubuntu16.04 (Deb)"を選択し、ダウンロード。  
ダウンロードしたファイルをダブルクリックするとソフトウェアインストーラが表示されるので、インストールを選択すると、無事にインストールされた  

` dpkg -l | grep cudnn `でインストールされてるかどうかが確認できる
